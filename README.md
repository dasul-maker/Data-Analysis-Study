## 🚀 실행 방법
1. 이 저장소를 클론(clone) 하거나 Colab에서 `.ipynb` 파일을 열어 실행합니다.
   ```bash
   git clone https://github.com/username/healthcare-project.git
   ```
2. 필요한 경우 data/ 폴더에 CSV 파일을 업로드하세요.
3. Colab에서 런타임 → 모두 실행을 선택하여 실행합니다.
